call of duty zombies
made by armin_iuf (2017)

There are 3 maps in this game, each with two bosses and a lot of rooms. At the beginning of the game, one of these 3 maps will randomly selected, each map has its own mysteries and strategies. Stay up to 40 rounds to complete each map. as the rounds pass, the game becomes more difficult, so be sure to use the map features like traps. to remove the door blocks if you have enough money you should hit them. The best thing to do first is turn on the power button so you can use the map features. Also available perks can make your character stronger.
